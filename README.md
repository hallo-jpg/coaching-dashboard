## Workflow

1. **Neue Woche:** Claude Code liest aktuelle KW → schreibt nächste KW → archiviert abgeschlossene
2. **FTP-Test:** Ergebnis mitteilen → Claude Code aktualisiert Zonen + Fortschritts-Log
3. **Workout-Feedback:** Status (✅/❌) + Notiz → Claude Code trackt Compliance
