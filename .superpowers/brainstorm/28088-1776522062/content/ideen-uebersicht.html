📲 Dashboard · Verfügbarkeit & Zugriff
📱
PWA – „Add to Home Screen"
Einfach
Dashboard.html wird zur Progressive Web App: Manifest + Service Worker hinzufügen. Stefan fügt es wie eine App zum iPhone-Homescreen hinzu. Kein Server, kein Deploy — lokale Datei bleibt die Quelle, nur die Darstellung ändert sich.
→ Sofortiger Zugriff vom Homescreen, funktioniert offline
🌐
Auto-Deploy auf Netlify / GitHub Pages
Mittel
Jedes Mal wenn der Coach Skill dashboard.html aktualisiert, pushed ein Git-Hook automatisch zu einem privaten Netlify/GitHub-Pages-Repo. Dashboard ist dann unter einer festen URL erreichbar — von überall, ohne lokalen Server.
→ Feste URL, immer aktuell, kein Öffnen von Dateien nötig
⚡
Live-Dashboard mit API-Backend
Komplex
Cloudflare Worker oder kleines Node-Backend; der Coach Skill schreibt JSON-Daten in KV/DB, das Dashboard liest live via fetch(). Daten wie Readiness, CTL/ATL, Wochenplan sind immer aktuell ohne HTML-Neugenerierung.
→ Echte Live-Daten, Dashboard trennt sich von Claude-Workflow
🤖 Coach Skill · Automation & Feedback-Loops
📬
Automatisches Weekly Briefing per E-Mail
Mittel
Jeden Montag früh sendet ein Cron-Job (Claude Code Cron) automatisch das Wochen-Briefing via Gmail MCP an Stefan. Kein manueller /coach-Aufruf nötig — der Plan kommt einfach im Postfach an.
→ Kein Aufwand um den Plan zu bekommen, Push statt Pull
🔄
Schneller Workout-Feedback-Loop
Einfach
Ein kurzes Template: „✅ LIT-2h | RPE 6 | Knie ok" — Claude parsed das, aktualisiert TSS Ist in kw16.md, passt bei Bedarf die Restwoche an und updated dashboard.html. Alles in einem Rutsch, kein ausführliches Briefing nötig.
→ Minimaler Aufwand für Stefan, maximale Aktualität
🧭
Interaktiver Saisonplanung-Wizard
Mittel
Post-Season: Ein geführter Dialog (5–6 Fragen) fragt nach Zielen, Events, verfügbarer Zeit und Schwächen. Danach generiert der Coach einen kompletten neuen Saisonplan inkl. periodisierung.md, CTL-Kurve und Ausblick.
→ Strukturierter Einstieg in neue Saison, keine leere Seite
🚨
Automatische Readiness-Alerts
Mittel
Claude prüft täglich via Cron intervals.icu Readiness. Fällt der Score unter 50 oder springt ein Krank-Risiko-Muster auf, sendet Claude automatisch einen Hinweis (Push-Notification oder E-Mail): „Heute kein Training empfohlen."
→ Proaktive Empfehlung ohne dass Stefan fragen muss
🧠 Coach Skill · Intelligenz & Tiefe
📈
Automatische Block-Skizze nach jeder Planung
Einfach
Nach jeder Wochenplanung generiert der Coach automatisch einen aktualisierten 4-Wochen-Ausblick und schreibt ihn direkt ins Dashboard. Heute nur auf Abruf — zukünftig immer aktuell ohne extra Schritt.
→ Dashboard zeigt immer den echten Vorausblick, kein manueller Trigger
🍽️
Fueling-Einkaufsliste
Einfach
Nach der Wochenplanung berechnet der Coach automatisch die Gesamtmengen: „Diese Woche brauchst du 480g Maltodextrin, 120g Fructose, 15g Salztabletten." Als einkaufbereite Liste, nicht als Tabelle.
→ Direkt umsetzbar, kein Nachrechnen aus 7 Fueling-Zeilen
📊
Monats-Retrospektive
Mittel
Ende jeden Monats: Claude analysiert alle archivierten KW-Dateien + intervals.icu-Daten und erstellt eine Retrospektive: Soll vs. Ist TSS, Zonen-Compliance, Trends, was gut lief, was angepasst werden sollte.
→ Strukturiertes Lernformat, Muster über Wochen hinweg sichtbar
